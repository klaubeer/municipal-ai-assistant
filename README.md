Development of an Artificial Intelligence assistant designed to simplify access to municipal public information.

The system allows citizens to query information using natural language about topics such as municipal laws, decrees, public services, administrative procedures, taxes, infrastructure works, and other government-related information, without needing to manually navigate complex public portals.

The solution uses Retrieval-Augmented Generation (RAG) to retrieve information from official sources, combining public data scraping, document processing, and LLM-based response generation.

The assistant is designed to reduce friction in accessing public information, enabling citizens to quickly obtain answers about municipal services and regulations.

In addition to the conversational assistant, the project includes an analytical dashboard that tracks user queries, helping identify recurring demands from the population and providing insights to improve transparency and efficiency in public service.

Desenvolvimento de um assistente baseado em Inteligência Artificial voltado ao acesso simplificado a informações públicas municipais.

O sistema permite que cidadãos consultem, por meio de linguagem natural, informações relacionadas a leis, decretos, serviços públicos e procedimentos administrativos, sem a necessidade de navegar manualmente por portais governamentais complexos.

A solução utiliza técnicas de RAG (Retrieval-Augmented Generation) para recuperar informações a partir de fontes oficiais, combinando raspagem de dados públicos, processamento de documentos e geração de respostas por LLM.

O assistente foi projetado para reduzir a fricção no acesso à informação pública, permitindo que munícipes obtenham respostas rápidas sobre temas como impostos, obras, serviços municipais, legislação e processos administrativos.

Além do assistente conversacional, o projeto inclui um painel analítico que monitora consultas realizadas, permitindo identificar demandas recorrentes da população e melhorar a transparência e eficiência do atendimento público.
