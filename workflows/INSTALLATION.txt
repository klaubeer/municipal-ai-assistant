## Instalação

Este projeto contém workflows prontos para importar no n8n.

Para utilizá-los:

1. Baixe o arquivo JSON do workflow na pasta `/workflows`.
2. Abra sua instância do n8n.
3. Clique em **Importar workflow**.
4. Selecione o arquivo JSON baixado.
5. Configure suas credenciais (Supabase, APIs, etc.).
6. Ative o workflow.

## Requisitos

- n8n (self-hosted ou cloud)
- Banco de dados Supabase
- Provedor de LLM (OpenAI ou compatível)
- Integração com plataforma de mensageria (opcional)
